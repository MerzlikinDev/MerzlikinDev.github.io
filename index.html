Теория

Нормальный алгоритм Маркова - модель вычислений, работающая со строками. Есть список правил вида левая → правая. На каждом шаге ищется самое левое вхождение левой части и заменяется на правую. Правила просматриваются по порядку сверху вниз.

Если правило содержит => (заключительное), работа останавливается сразу после замены.
Если ни одно правило не применимо - алгоритм завершается.

Пример 1: удаление повторов букв

aa -> a
bb -> b
cc -> c
Вход: abbbcaa Результат: abca

Пример 2: удвоение каждого символа

a -> aa
b -> bb
Вход: ab Результат: А резульатата не будет, т.к программа зациклится

Пример 3: удаление всех 'x'

x -> (пустая правая часть)
Вход: xxaxxb Результат: ab

Правила в эмулятор загружаются через файл .txt или вводятся в поле. Формат: левая_часть -> правая_часть или левая_часть => правая_часть
